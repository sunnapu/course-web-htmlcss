width: auto vs width: 100%

reference from http://www.456bereastreet.com/lab/width-auto/

I am a container with width 600px.

width: auto
content width = parent width - horizontal margin width - horizontal border width - horizontal padding width
my containing width = parent width

width: 100%;
content width = parent width - horizontal border width - horizontal padding width my containing width = horizontal margin width + parent width

width: 100%;
content width = parent width my containing width = horizontal margin width + horizontal border width + horizontal padding width + content width
