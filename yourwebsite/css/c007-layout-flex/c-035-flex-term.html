Why need flexbox

CSS centering and CSS layout require work (float, inline-blocks, even tables, css tricks) -> many CSS grid frameworks -> standard way to do CSS layout -> flexbox

Base concept

Flex container, Flex items. Every direct child is a flex item which means outside of container and inside of flex item rendered as usual.
