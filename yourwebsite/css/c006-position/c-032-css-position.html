Floating elements like boats on the sea. We tell browser where to float, the browser makes final decision where to put it without overlapping with other floating elements. Float is used when we want to surround text with image, but people also use it for page structure layout before 'flex' coming into CSS.

clear: when floating elements and non-floating elements sometimes be top of each other.'clear' is for control surrounding. This is why when use float to draw a layout, we most of time have to use 'clear' together.
