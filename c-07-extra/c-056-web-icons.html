Concepts:

  • Web icons technically have no differences with fonts, they are all shapes.
  • Web icons are scalable, not like gif or png, they lost details when you make them bigger.
  • Web icons support transparency too.
  • Web icons only can have single color or gradient, if need more color we use images.

Font formats

  • ttf: true type format
  • otf: open type format
  • ttf and otf can be used for desktop fonts
  • svg: scalable vector graphics
  • eot: embedded open type used only for ie
  • woff: web open font format used for all main browsers

How it works:

  • Font awesome defines its own font 'FontAwesome'
  • Font awssome css import the font
  • Font awssome css set 'font-family:FontAwesome;' for the class 'fa'
  • Font awesome css set unicode character &#xf0e0; (in css \f0e0) for the class 'fa-envelope:before'
